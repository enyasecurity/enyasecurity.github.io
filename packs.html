Packs cybersécurité

Choisissez le pack adapté à votre besoin

Pack Web Check

Un contrôle de votre présence en ligne pour identifier ce que votre site expose publiquement.

Idéal pour obtenir une vue claire de votre présence en ligne et corriger les points sensibles immédiatement.

Pack Analyse Sécurité Web

Une analyse complète de votre site pour détecter les failles les plus courantes.

Je vérifie ce que votre site expose publiquement, j’analyse les failles potentielles et je vous fournis un rapport clair avec les actions à mettre en place.

Pack Sécurité Web Avancé

Un audit approfondi pour identifier les failles importantes et les risques qui pourraient vous impacter.

Je vérifie les points faibles du site, effectue des tests de sécurité non intrusifs et vous remets un rapport clair avec les recommandations essentielles.

Pack Analyse Sécurité WordPress

Une analyse dédiée à WordPress pour repérer les failles des plugins, thèmes et versions.

Je vérifie l’état de votre WordPress : plugins, thèmes, fichiers exposés, versions et éléments sensibles. Le rapport final vous indique les risques et les actions à effectuer.

Pack Audit Web PME

Une vision globale de la sécurité du site et de l’organisation pour mieux protéger l’entreprise.

Une analyse complète de votre site et de vos pratiques : exposition en ligne, vulnérabilités techniques, tests de sécurité, et simulation de phishing. Le rapport final présente les risques identifiés et un plan d’action pour renforcer votre sécurité.

Pack Sensibilisation & Phishing

Aidez vos équipes à reconnaître les menaces et éviter les erreurs humaines.

Une simulation de phishing pour mesurer le niveau de vigilance, suivie d’une session de sensibilisation et d’un rapport complet avec statistiques et recommandations.

Pack Incident (Diagnostic d’urgence)

Une analyse rapide pour comprendre l’origine d’un problème ou d’une suspicion d’incident.

Je réalise une vérification express : logs serveur, fichiers critiques, redirections, DNS et traces d’activité malveillante. Le rapport final présente la cause la plus probable et les actions urgentes à appliquer.
